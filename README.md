This is a simple and clean Tic Tac Toe game built using HTML, CSS, and JavaScript.
It includes both 2-Player mode and Player vs AI mode, along with 3 difficulty levels.

Game Modes :-
1. Two Player Mode
Both players play on the same device using X and O.

2. Player vs AI
 
There are three difficulty levels:

Easy :
Makes random moves, Very easy to win

Medium : 
Mix of smart and random moves, Gives a balanced challenge

Hard (Minimax) : 
Uses the minimax algorithm, Predicts all possible outcomes, Always makes the best move, Almost impossible to beat

Features :-
Neon-themed UI
Smooth button animations
Sound effects (click, win, lose, draw)
Highlight for winning combination
Fully responsive layout (mobile + desktop)
Human always plays first

Technologies Used :-
HTML5
CSS3 
JavaScript

Game logic :-
AI logic (minimax)
DOM updates
Event handling
Sound control

Deployment :-
The project is deployed using Netlify for fast and easy static hosting.
